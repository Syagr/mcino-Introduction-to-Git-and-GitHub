# Simple Interest Calculator

This project contains a simple interest calculator script.

## Description
The calculator computes **simple interest** based on user input:
- Principal amount (P)
- Rate of interest (R)
- Time period in years (T)

## Formula
Simple Interest (SI) is calculated as:

SI = (P × R × T) / 100
